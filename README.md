# Monthly Mileage Tracker

A Python command-line application that stores mileage information for 12 months.

The program uses lists, functions, loops, conditional statements, function arguments, and return values. Users can populate monthly records, search for a specific month, and identify the months with the smallest and largest mileage totals.

## Features

- Stores 12 month names and mileage values
- Searches for mileage by month
- Finds the month with the lowest mileage
- Finds the month with the highest mileage
- Uses a menu-driven interface
- Uses functions, arguments, return values, and lists

- ## Python Concepts Used

- Lists
- Functions
- Function arguments
- Return values
- While loops
- Conditional statements
- User input
- Searching and comparison logic
